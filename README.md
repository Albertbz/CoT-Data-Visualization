# TypeScript Project (minimal scaffold)

This is a minimal TypeScript project scaffold created for the workspace.

Quick start

1. Install dependencies:

   npm install

2. Development (run TypeScript directly):

   npm run dev

3. Build and run compiled output:

   npm run build
   npm start

Files created

- `package.json` — scripts and devDependencies
- `tsconfig.json` — TypeScript compiler options
- `src/index.ts` — example entrypoint

Notes

- Change TypeScript versions or add linting/testing as needed.
